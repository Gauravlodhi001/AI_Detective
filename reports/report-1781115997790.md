### 1. [Critical] Hardcoded AWS Access Key ID

* **Rule ID:** `hardcoded-secret-aws-access-key-id`
* **File:** `vuln_snippet.js` (Line 3)
* **CWE:** [CWE-798](https://cwe.mitre.org/data/definitions/798.html)
* **OWASP Category:** A02:2021-Cryptographic Failures

**Description:**
A hardcoded AWS Access Key ID was detected. Anyone with access to this key can potentially make API calls to your AWS account. (Found: `AKIA************MPLE`)

**Vulnerable Snippet:**
```
const AWS_ACCESS_KEY_ID = "AKIAIOSFODNN7EXAMPLE";
```

**Remediation Suggestion:**
Remove the hardcoded secret from your code. Store secrets in environment variables or use a secret management service (like HashiCorp Vault, AWS Secrets Manager, or dotenv files configured in .gitignore).

**Remediation Diff:**
```diff

==== SUGGESTED FIX

>>>>
```
